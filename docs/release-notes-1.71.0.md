# Driftile 1.71.0

Driftile 1.71.0 makes the optional Overview's keyboard and structured-search
controls discoverable without adding another global shortcut.

## Highlights

- Press `F1` to open a compact keyboard reference inside the Overview.
- Review navigation, activation, closure, search editing, structured-search
  fields, and search operators in one adaptive panel.
- Close the panel with `F1`, `Escape`, or its pointer button.
- Keep the panel usable on small scenes through bounded scrolling while all
  background Overview input remains inactive.
- Discover the existing F1 action through a passive hint that hides during
  search and while help is open.

## Compatibility

KDE Plasma and KWin 6.7 or newer are required. The Overview help is session-only
and adds no global binding, setting, timer, animation, layout write, or
persistence schema. Logical persistence remains v4.
