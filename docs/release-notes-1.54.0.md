# Driftile 1.54.0

Driftile 1.54.0 makes minimized members of tabbed columns actionable in the
optional Overview.

## Highlights

- Reach an existing minimized member tab with the pointer, arrow keys, Tab, or
  search.
- Restore and focus the exact selected window with a click, `Enter`, `Return`,
  or `Space`.
- Close an exact closeable minimized tab with middle click or `Delete` without
  restoring it first.
- Find minimized tabs with `minimized`, combined with title, application,
  `urgent`, or `attention` terms.
- Keep ordinary selected tabs inert and keep minimized windows outside every
  drag-and-drop path.

## Compatibility

KDE Plasma and KWin 6.7 or newer are required. The feature belongs only to the
optional Overview effect and uses public KWin minimized and active-window
state. The main layout script, settings, actions, default bindings, transition
effect, and logical persistence v4 are unchanged.
