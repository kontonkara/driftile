# Driftile 1.73.0

Driftile 1.73.0 makes an overflowing Overview keyboard reference usable without
leaving the modal panel.

## Highlights

- Scroll with `Up`/`Down`, `Page Up`/`Page Down`, or `Home`/`End`.
- Scroll with a mouse or touchpad wheel while retaining touch drag.
- Clamp every path to the available content and safely consume input when the
  complete reference already fits.
- Keep all background Overview selection, focus, window, and desktop actions
  inactive while help is open.

## Compatibility

KDE Plasma and KWin 6.7 or newer are required. The interaction is session-only
and adds no global binding, setting, timer, animation, layout write, or
persistence schema. Logical persistence remains v4.
